## 1. Macro Narrative: Starknet's Identity Shift

Starknet has rebranded itself as **"The Bitcoin DeFi Layer"** -- a dramatic pivot from its original Ethereum L2 positioning. The official tagline is now: *"Supercharged by ZK Tech."*

**Key positioning pillars (as of March 2026):**

- First L2 to settle on BOTH Bitcoin and Ethereum
- Native BTC staking (WBTC, tBTC, SolvBTC supported)
- Privacy-first architecture (STRK20, strkBTC)
- The "Bitcoin execution layer" -- enabling complex applications on BTC
- S-two prover: fastest prover for client-side ZK proving on everyday devices

**This is what's generating the most conversation.** Starknet is trying to own the "Bitcoin DeFi" narrative the same way Solana owned "fast and cheap" and Base owned "consumer crypto."

## 3. Gaming: Cartridge + Dojo Ecosystem

### Cartridge (cartridge.gg)

**Positioning:** "Onchain Gaming Platform for Studios"

**Three pillars:**

1. **Controller** (Play with)
   - Gaming-focused smart contract wallet
   - Passkey authentication (passwordless, one-click onboarding)
   - Session keys (eliminate transaction popups during gameplay)
   - Cartridge Paymaster (free/gasless transactions for players)
   - Multi-signer: Passkeys, password, social login, external wallets
   - Self-custodial embedded wallets
   - Full theme customization for game branding
   - Achievement system + universal player identity
   - Monetization: multi-chain crypto + fiat, Starterpack bundles, Booster packs, NFT marketplace

2. **Slot** (Scale with)
   - Horizontally scalable execution sharding
   - Ephemeral and persistent rollups
   - Low-latency execution contexts with fixed costs

3. **Arcade** (Discover with)
   - Central hub for onchain games
   - Player and developer discovery platform

### Dojo Engine

Provable game engine providing a comprehensive framework for building verifiable games and autonomous worlds.

**Notable Games Built with Dojo:**
- **Eternum** (BibliothecaDAO/Realms Autonomous World) -- the flagship
- **Beer Baron** (Cartridge) -- onchain simulation, 10-10,000 players
- **Dope Wars: Roll Your Own** (Cartridge) -- classic Drug Wars adaptation
- **Drive AI** (Cartridge) -- onchain driving simulator with neural network
- **Pistols at Ten Blocks** -- dueling game
- **PixeLAW** -- pixel grid Autonomous World with coexisting games
- **Mancala** -- strategy board game
- **The Cave** -- roguelike deckbuilder
- **zDefender** -- tower defense
- **StarkLand** -- multiplayer online simulation (FOMOSLG)
- **Stark Lander** -- Moon landing game
- **Emoji Man** -- Pac-man inspired
- **Chess Dojo** -- chess

**Key library:** Origami -- Cairo primitives for onchain gaming

### What Cartridge is Enabling

Cartridge solves the "onboarding cliff" for onchain gaming:
1. No wallet setup (Passkey = fingerprint/face)
2. No gas fees (Paymaster covers it)
3. No transaction popups (Session keys)
4. Real money rewards + player-owned economies
5. Interoperable assets across games

**The pattern:** Gaming is Starknet's strongest consumer vertical. Cartridge+Dojo is the most mature onchain gaming stack in any L2 ecosystem.

## 6. AVNU Paymaster -- Gasless Transaction Use Cases

**AVNU offers a complete trading toolkit:**

1. **Swap** -- Solver-optimized, NP-hard routing problems solved in real-time
2. **DCA** -- Recurring token purchases with automated execution
3. **Paymaster** -- Users pay gas in ANY token, or sponsors cover gas entirely
4. **Markets API** -- Real-time prices, liquidity, volume
5. **Token List** -- Verified metadata and contract addresses

**Paymaster creative uses:**
- Gaming: Cartridge uses paymaster so players never see gas
- Consumer apps: Ready wallet abstracts gas away
- DeFi: Protocols sponsor gas for first-time users
- Starknet blog post (May 2025): "Paymaster: The Secret to Making dApps Feel Like Web2"

**AVNU Paymaster enables:**
- Gasless transactions where the dApp pays
- Users pay gas in any token (not just ETH/STRK)
- SDK integration in ~5 minutes (TypeScript)
- 50+ protocols already integrated

**The insight:** Paymaster is infrastructure, not a product. The winning apps are the ones that use it invisibly. The user never knows gas exists.

## 7. Social Login on Starknet

### Current Solutions

1. **zKey** -- zkLogin using zero-knowledge proofs, social account login (Google, Apple), self-custodial
2. **Cartridge Controller** -- Passkey authentication (biometric), not traditional "social login" but achieves the same frictionless onboarding
3. **Coin98** -- Social Log-in Wallet feature
4. **Dfns** -- WaaS (Wallets-as-a-Service) platform for developer-managed onboarding

### The Gap

Social login on Starknet exists but is fragmented. No single dominant solution like Privy/Dynamic on EVM chains. This is an opportunity:
- zKey is the closest pure social login play
- Cartridge Controller is the best UX but gaming-focused
- No universal "connect with Google/Apple/X" SDK for all Starknet dApps

## 10. Emerging Gaps & Opportunities

### High-Conviction Opportunities

1. **Privacy-First Consumer Apps**
   - STRK20 just launched (4 days ago). No consumer apps built on top yet.
   - Private payments, private DeFi, private social -- all now possible.
   - First mover advantage is massive.

2. **Bitcoin DeFi Aggregation**
   - Multiple BTC wrappers (WBTC, tBTC, SolvBTC, LBTC, strkBTC) creating fragmentation
   - Opportunity: unified BTC DeFi experience that abstracts wrapper complexity
   - Starkzap is positioned here but execution matters

3. **Social/Consumer Gaming Beyond Cartridge**
   - Cartridge owns infrastructure but the "Roblox of onchain gaming" doesn't exist yet
   - Casual, mobile-first games with real money rewards
   - Games that don't look or feel like crypto

4. **Cross-Chain Intent Fulfillment**
   - Near Intents is on Starknet now (listed #1 in ecosystem)
   - Intent-based UX where users say "what" not "how"
   - AI agent integration for intent expression

5. **Yield Optimization / Auto-Strategies**
   - BTC staking + LSTs + DeFi composability = complex yield stacking
   - No dominant "Yearn for Starknet BTC" yet
   - Looping strategies (Troves) exist but are manual

6. **Developer Tooling for Paymaster Integration**
   - Paymaster exists but integration is still friction
   - Opportunity: higher-level abstractions, plug-and-play gasless SDKs
   - AVNU SDK is good but Starkzap could own "gasless DeFi in 3 lines of code"

### Underserved Verticals

7. **Meme/Fun Apps**
   - Almost no meme ecosystem on Starknet compared to Solana/Base
   - No dominant meme coin launchpad
   - No "pump.fun" equivalent
   - Gap: fun, viral, low-barrier-to-entry token experiences

8. **Social Features / Community Tools**
   - No Starknet-native social protocol (like Farcaster on Base/OP)
   - Cartridge has achievements/identity but it's gaming-only
   - Opportunity: universal onchain identity + reputation + social graph

9. **Real-World Asset (RWA) / Payments**
   - Circle USDC/EURC are live but no compelling payment apps
   - Ready wallet is trying ("alternative to your bank") but early
   - Opportunity: payroll, invoicing, P2P payments with privacy (STRK20)

10. **AI + Blockchain**
    - Near Intents mentions AI agents explicitly
    - No dominant AI-agent framework on Starknet
    - Opportunity: AI agents that use Starknet for settlement, trading, or gaming
